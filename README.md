# TWStockPicker

Purpose: 根據財務報表數據及某些特定條件篩選出適合投資的股票。

Usage: ```python examine_ipodate.py```

### Feature：
- 篩選條件：
  - 根據黃國華的著作「財務自由的講堂+財務自由的世界」中所運用的篩選條件為基礎
  - 已完成：篩選掉未滿兩年的股票
- 使用csv檔輸入要進行篩選的股票
